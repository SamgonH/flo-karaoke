## 프로젝트 목표

FLO 앱 사용자가 외국곡의 노래방 번호(TJ/KY)를 한글로 쉽게 검색하고, 즐겨찾기로 저장·관리까지 할 수 있는 노래방 번호 검색 툴

## 타겟 사용자

FLO 앱 사용자 중 노래방을 즐기는 사람 — 특히 외국곡(J-POP, 팝, 중국 음악 등)을 자주 부르는 유저

## 주요 기능

### 1. 노래방 번호 검색
- 곡 제목 또는 아티스트명으로 검색
- **한글 입력 지원** — "요루시카"로 검색해도 ヨルシカ 곡 결과 표시
- TJ미디어, KY금영 두 기계의 번호를 함께 표시
- 검색 결과에 앨범 커버 이미지 표시

### 2. 노래 미리듣기
- 검색 결과에서 FLO 앱 내 미리듣기 연동 (앱 스킴 활용)

### 3. 즐겨찾기 & 나만의 폴더
- 마음에 드는 곡을 즐겨찾기로 저장 (로그인 필요)
- 나만의 폴더를 만들어서 분류 관리 (예: "친구랑 부를 곡", "혼자 연습할 곡")

### 4. 최근 검색 기록
- 이전에 검색한 곡 목록 확인
- 기기에 저장 (로컬 스토리지) — 비로그인도 사용 가능

### 5. 나라별 인기곡 순위
- 사용자들의 검색 횟수를 집계해 인기 순위 표시
- 카테고리: 팝송 🇺🇸 / 일본 🇯🇵 / 중국 🇨🇳 3개 탭
- 순위 항목에서 바로 번호 확인 및 즐겨찾기 가능

## 사용 환경

- 플랫폼: FLO 앱 WebView
- 로그인 필요 여부: 일부 기능 필요 (즐겨찾기·폴더는 로그인 기반, 검색·최근기록·순위는 비로그인 가능)

## 데이터 저장

- Supabase 사용: **사용**
- 저장할 데이터:
  - 노래방 곡 DB (TJ/KY 번호, 곡명, 아티스트, 한글 검색어 매핑, 장르/국가 분류)
  - 곡별 검색 횟수 (순위 집계용)
  - 사용자 즐겨찾기
  - 사용자 폴더

## 외부 연동

- FLO 앱 토큰 인증 (로그인 상태 연동)
- FLO 음악 미리듣기 앱 스킴 연동
- TJ미디어 / KY금영 사이트 크롤링 (초기 데이터 수집)

### 6. 공유 리스트 (같이 편집)
- 오늘 노래방에서 부를 곡 목록을 만들고 링크로 공유
- 링크 받은 친구도 로그인 없이 곡 추가·삭제 가능 (같이 편집)
- 리스트에는 TJ/KY 번호가 함께 표시되어 바로 신청 가능
